Подключение Supabase

Эти поля можно заполнить прямо здесь для текущего браузера. Чтобы облако работало на всех телефонах и устройствах, продублируйте значения в файле cloud-config.js перед загрузкой сайта на хостинг.

Telegram уведомления

Telegram включается через Supabase Edge Function. Сайт вызывает функцию, а токен бота хранится в secrets Supabase, а не в браузере.

Действия по облаку

Что дальше

  • Выполнить SQL из файла SUPABASE_SETUP.sql.
  • Заполнить cloud-config.js и загрузить сайт на хостинг.
  • Для Telegram задеплоить функцию supabase/functions/telegram-notify.
  • После этого заявки будут сохраняться в Supabase и станут доступны с разных телефонов.
